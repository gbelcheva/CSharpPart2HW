<a id="user-content-problem-4-binary-search" class="anchor" href="#problem-4-binary-search" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 4. Binary search</h3>

<ul class="task-list">
<li>  Write a program, that reads from the console an array of <code>N</code> integers and an integer <code>K</code>, sorts the array and using the method <code>Array.BinSearch()</code> finds the largest number in the array which is â¤ <code>K</code>. </li>

<h3>
<a id="user-content-problem-5-sort-by-string-length" class="anchor" href="#problem-5-sort-by-string-length" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 5. Sort by string length</h3>

<ul class="task-list">
<li>  You are given an array of strings. Write a method that sorts the array by the length of its elements (the number of characters composing them).</li>
</ul>

<h3>
<a id="user-content-problem-6-matrix-class" class="anchor" href="#problem-6-matrix-class" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 6.* Matrix class</h3>
