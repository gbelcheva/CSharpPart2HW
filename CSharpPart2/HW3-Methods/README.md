    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-homework-methods" class="anchor" href="#homework-methods" aria-hidden="true"><span class="octicon octicon-link"></span></a>Homework: Methods</h1>

<h3>
<a id="user-content-problem-1-say-hello" class="anchor" href="#problem-1-say-hello" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 1. Say Hello</h3>

<ul class="task-list">
<li>  Write a method that asks the user for his name and prints <code>“Hello, &lt;name&gt;”</code>
</li>
<li>  Write a program to test this method.</li>
</ul>

<p><em>Example:</em></p>

<table>
<thead>
<tr>
<th align="center">input</th>
<th align="center">output</th>
</tr>
</thead>
<tbody>
<tr>
<td align="center">Peter</td>
<td align="center">Hello, Peter!</td>
</tr>
</tbody>
</table>

<h3>
<a id="user-content-problem-2-get-largest-number" class="anchor" href="#problem-2-get-largest-number" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 2. Get largest number</h3>

<ul class="task-list">
<li>  Write a method <code>GetMax()</code> with two parameters that returns the larger of two integers.</li>
<li>  Write a program that reads <code>3</code> integers from the console and prints the largest of them using the method <code>GetMax()</code>.</li>
</ul>

<h3>
<a id="user-content-problem-3-english-digit" class="anchor" href="#problem-3-english-digit" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 3. English digit</h3>

<ul class="task-list">
<li>  Write a method that returns the last digit of given integer as an English word.</li>
</ul>

<p><em>Examples:</em></p>

<table>
<thead>
<tr>
<th align="center">input</th>
<th align="center">output</th>
</tr>
</thead>
<tbody>
<tr>
<td align="center">512</td>
<td align="center">two</td>
</tr>
<tr>
<td align="center">1024</td>
<td align="center">four</td>
</tr>
<tr>
<td align="center">12309</td>
<td align="center">nine</td>
</tr>
</tbody>
</table>

<h3>
<a id="user-content-problem-4--appearance-count" class="anchor" href="#problem-4--appearance-count" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 4.  Appearance count</h3>

<ul class="task-list">
<li>  Write a method that counts how many times given number appears in given array.</li>
<li>  Write a test program to check if the method is workings correctly.</li>
</ul>

<h3>
<a id="user-content-problem-5-larger-than-neighbours" class="anchor" href="#problem-5-larger-than-neighbours" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 5. Larger than neighbours</h3>

<ul class="task-list">
<li>  Write a method that checks if the element at given position in given array of integers is larger than its two neighbours (when such exist).</li>
</ul>

<h3>
<a id="user-content-problem-6-first-larger-than-neighbours" class="anchor" href="#problem-6-first-larger-than-neighbours" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 6. First larger than neighbours</h3>

<ul class="task-list">
<li>  Write a method that returns the index of the first element in array that is larger than its neighbours, or <code>-1</code>, if there’s no such element.</li>
<li>  Use the method from the previous exercise.</li>
</ul>

<h3>
<a id="user-content-problem-7-reverse-number" class="anchor" href="#problem-7-reverse-number" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 7. Reverse number</h3>

<ul class="task-list">
<li>  Write a method that reverses the digits of given decimal number.</li>
</ul>

<p><em>Example:</em></p>

<table>
<thead>
<tr>
<th align="center">input</th>
<th align="center">output</th>
</tr>
</thead>
<tbody>
<tr>
<td align="center">256</td>
<td align="center">652</td>
</tr>
<tr>
<td align="center">123.45</td>
<td align="center">54.321</td>
</tr>
</tbody>
</table>

<h3>
<a id="user-content-problem-8-number-as-array" class="anchor" href="#problem-8-number-as-array" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 8. Number as array</h3>

<ul class="task-list">
<li>  Write a method that adds two positive integer numbers represented as arrays of digits (each array element <code>arr[i]</code> contains a digit; the last digit is kept in <code>arr[0]</code>).</li>
<li>  Each of the numbers that will be added could have up to <code>10 000</code> digits.</li>
</ul>

<h3>
<a id="user-content-problem-9-sorting-array" class="anchor" href="#problem-9-sorting-array" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 9. Sorting array</h3>

<ul class="task-list">
<li>  Write a method that return the maximal element in a portion of array of integers starting at given index.</li>
<li>  Using it write another method that sorts an array in ascending / descending order.</li>
</ul>

<h3>
<a id="user-content-problem-10-n-factorial" class="anchor" href="#problem-10-n-factorial" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 10. N Factorial</h3>

<ul class="task-list">
<li>  Write a program to calculate <code>n!</code> for each <code>n</code> in the range [<code>1..100</code>].</li>
</ul>

<p><em>Hint: Implement first a method that multiplies a number represented as array of digits by given integer number.</em></p>

<h3>
<a id="user-content-problem-11-adding-polynomials" class="anchor" href="#problem-11-adding-polynomials" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 11. Adding polynomials</h3>

<ul class="task-list">
<li>  Write a method that adds two polynomials.</li>
<li>  Represent them as arrays of their coefficients.</li>
</ul>

<p><em>Example:</em></p>

<p>x<sup>2</sup> + 5 = <code>1</code>x<sup>2</sup> + <code>0</code>x + <code>5</code>   =&gt;  {<code>5</code>, <code>0</code>, <code>1</code>}</p>

<h3>
<a id="user-content-problem-12-subtracting-polynomials" class="anchor" href="#problem-12-subtracting-polynomials" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 12. Subtracting polynomials</h3>

<ul class="task-list">
<li>  Extend the previous program to support also subtraction and multiplication of polynomials.</li>
</ul>

<h3>
<a id="user-content-problem-13-solve-tasks" class="anchor" href="#problem-13-solve-tasks" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 13. Solve tasks</h3>

<ul class="task-list">
<li>  Write a program that can solve these tasks:

<ul class="task-list">
<li>  Reverses the digits of a number</li>
<li>  Calculates the average of a sequence of integers</li>
<li>  Solves a linear equation <code>a * x + b = 0</code>
</li>
</ul>
</li>
<li>  Create appropriate methods.</li>
<li>  Provide a simple text-based menu for the user to choose which task to solve.</li>
<li>  Validate the input data:

<ul class="task-list">
<li>  The decimal number should be non-negative</li>
<li>  The sequence should not be empty</li>
<li>  <code>a</code> should not be equal to <code>0</code>
</li>
</ul>
</li>
</ul>

<h3>
<a id="user-content-problem-14-integer-calculations" class="anchor" href="#problem-14-integer-calculations" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 14. Integer calculations</h3>

<ul class="task-list">
<li>  Write methods to calculate <code>minimum</code>, <code>maximum</code>, <code>average</code>, <code>sum</code> and <code>product</code> of given set of integer numbers.</li>
<li>  Use variable number of arguments.</li>
</ul>

<h3>
<a id="user-content-problem-15-number-calculations" class="anchor" href="#problem-15-number-calculations" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 15.* Number calculations</h3>

<ul class="task-list">
<li>  Modify your last program and try to make it work for any number type, not just integer (e.g. decimal, float, byte, etc.)</li>
<li>  Use generic method (read in Internet about generic methods in C#).</li>
</ul>
